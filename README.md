# ❤️ Heart With Touch

An interactive HTML5 Canvas animation that creates beautiful glowing heart bursts wherever you click, touch, or drag. Designed for both desktop and mobile devices, this project delivers colorful heart particle effects with smooth animations and responsive controls.

## ✨ Features

* ❤️ Heart burst effect on click or touch
* 🌈 Random colorful glowing hearts
* 🖱️ Mouse support for desktop users
* 📱 Touch support for mobile devices
* ✨ Continuous heart generation while holding
* 🎨 Smooth fade-out and particle animations
* ⚡ Lightweight and dependency-free
* 🖥️ Responsive fullscreen canvas

## 🎮 How to Use

### Desktop

* Click anywhere to create a heart burst.
* Hold and drag the mouse to continuously generate hearts.

### Mobile

* Tap anywhere on the screen to create a heart burst.
* Touch and drag to create a stream of colorful hearts.
